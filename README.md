# 📊 Telecom Customer Churn Analysis Dashboard

## 📌 Project Overview
This project analyzes telecom customer churn data to identify patterns and factors that lead to customer churn. The analysis was performed using Python and Power BI, and an interactive dashboard was created to provide business insights.

---

## 🎯 Objectives
- Identify key factors affecting customer churn
- Calculate churn rate and customer retention rate
- Analyze churn by contract type, internet service, tenure, and payment method
- Build an interactive Power BI dashboard for business decision-making

---

## 🛠 Tools & Technologies Used
- Python
- Pandas
- Matplotlib
- Power BI
- DAX
- Data Cleaning
- Data Visualization

---

## 📈 Key Insights
- Overall churn rate is **26.54%**
- Customers with **Month-to-Month contracts have the highest churn**
- **Fiber Optic users have higher churn rates**
- **New customers (low tenure) are more likely to churn**
- Customers with **higher monthly charges have higher churn probability**

---

## 📊 Dashboard Features
- KPI Cards (Total Customers, Churn Rate, Stayed Rate, Joined Rate)
- Churn analysis by Contract Type
- Churn analysis by Internet Type
- Churn analysis by Payment Method
- Churn analysis by Tenure Group
- Churn analysis by Monthly Charges
- Interactive filters using slicers
